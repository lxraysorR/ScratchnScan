# Demo Script (MVP)

1. Open app to "Scan or Enter UPC".
2. Enter UPC manually.
3. Show success path with identified product.
4. Repeat with unknown UPC.
5. Show fallback: upload front + ingredients label photos.
6. Run AI understanding + homemade alternative.
7. Present result sections and disclaimer.
8. Tap "Scan another item" to restart loop.

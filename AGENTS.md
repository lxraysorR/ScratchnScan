# ScratchNScan Agent Instructions

## Mission
ScratchNScan converts packaged foods into simple homemade alternatives.

## Current phase
Bootstrap only. Do not implement full product code in this phase.

## MVP guardrails
Only build toward this loop:
1. UPC scan/manual entry OR photo label input
2. Product understanding
3. Homemade alternative output

## Out of scope for MVP
- subscriptions
- pantry inventory
- meal planning
- shopping/Instacart flows
- billing/payments
- advanced nutrition scoring
- full account system
- release operations

Track out-of-scope ideas in `docs/BACKLOG.md`.

## Working style
- Keep changes additive and non-destructive.
- Prefer clear docs/contracts before implementation.
- Keep prompts and agent specs synchronized with docs.
